## How to add new blog posts
1. Use http://prose.io
2. Open your project
3. Write new post
4. Put some images
5. Edit Metadata
6. Click Save

## How to get image url from Flickr
1. Copy-paste the image url or set url
2. Go to http://restlesspoint.com/admin/flickr/
3. Paste the Url
4. Copy paste the needed image url (small, full, etc)
